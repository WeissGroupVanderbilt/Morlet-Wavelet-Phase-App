Morlet Wavelet Phase Application

This application implements the Morlet Wavelet Phase algorithm, developed primarily to measure frequency changes of Fabry Perot fringes in thin film optical reflectance spectra, to enable low limits of detection for optical thin film sensors. However, it can be used in any application where low noise measurement of frequency changes of an approximately sinusoidal signal are required.

If you use this application, please cite the following paper which introduced the algorithm: Ward, S. J.; Layouni, R.; Arshavsky-Graham, S.; Segal, E.; Weiss, S. M. Signal Processing Techniques to Reduce the Limit of Detection for Thin Film Biosensors. arXiv:2103.07524 (2021). Please note that this application is for non-commercial use only.
